## Instructions to run

* Pre-requisites:
	-  Qt
	-  Conan
	-  GCC
	-  CMake(Preferred)
<br>

* How to install/build (with cmake):
	$ mkdir build
	$ cd build
	$ cmake ..
	$ cmake --build . --target all
