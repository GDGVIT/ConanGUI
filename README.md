## Instructions to run

* Pre-requisites:
	-  Qt
	-  Conan
	-  GCC
	-  CMake(Preferred)
	-  POSIX compliant(unix-like) OS.
<br>

* How to install/build (with cmake):  

  From the cloned directory,  
```
$ mkdir build  
$ cd build  
$ cmake ..  
$ cmake --build . --target all  
$ ./Barbarian
```
